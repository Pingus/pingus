What is Pingus?

Pingus is an open source (GPL license) game which is a clone of the video game Lemmings.

You can find more information at the project's Home Page.

What is this page for, then?

This wiki was created because Pingus' Home Page is quite a bit out of date, and it's tough to find up to date information on the project.

Development work

If you plan on doing any development work on Pingus, then you should sign up for the pingus-devel mailing list and the pingus-cvs mailing list to get the latest information. For quick questions, you might have better luck in IRC channels #pingus or #clanlib at irc.freenode.net:6667
